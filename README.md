#####一、使用UIImageView(感觉很野路子)
 1. 优点
  1. 设置UIImageView时，可以设置不同的专场显示效果
 2. 缺点
  1. 由于UIImageView的专场过程不可控，捕捉不到或无法控制专场过程。
#####二、使用UICollectionView
 1. 优点
  1. 可以捕捉, 控制专场过程。效果更好。
 2. 缺点
  1. 没有UIImageView 那样多的专场效果。(但这不重要)
  2. 当活动到最后，或者第一个 Cell 时会出现一些问题(后面会讲到)。
